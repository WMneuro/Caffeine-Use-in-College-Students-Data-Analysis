Caffeine Use in College Students Survey Analysis 
Statistical analysis and data visualization of survey responses using python.

This project analyzes survey data on caffeine use in distinct gender populations in addition to examining other factors mediating caffeine use. 

Repository Structure:
Caffeine_Project_Figures/ - generated figures  
Caffeine_Project_Analysis.py - main analysis script  

Methods:
The script tests statistical assumptions and runs appropriate statistical tests.
The script visualizes the data.

Installation:
pip install -r requirements.txt

Running the Analysis:
python Caffeine_Project_Analysis.py
